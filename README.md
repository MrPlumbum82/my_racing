##How to run:

###Linux:
 1. Install opengl, glut and soil libraries
 ```bash
  sudo apt-get install freeglut3-dev 
 ```

 2. Clone the repository
 
 ```bash
  git clone https://github.com/MrPlumbum82/my_racing
 ```

 3. Compile
 
 ```bash
  g++ -std=c++11 asteroids.cpp -lGL -lGLU -lglut 
 ```
 
 4. Execute
 
 ```bash
  ./a.out
 ```
